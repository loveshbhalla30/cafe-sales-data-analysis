# Cafe Sales Data Analysis

This project analyzes cafe sales data using SQL, Excel Power Query and Power BI.

## Tools Used
- SQL (PostgreSQL)
- Excel (Power Query)
- Power BI
- DAX

## Project Workflow
1. Imported messy cafe sales dataset
2. Cleaned data using SQL and Excel Power Query
3. Handled missing values and inconsistent categories
4. Created DAX measures for KPIs
5. Built an interactive Power BI dashboard

## Dashboard Insights
- Salad generates the highest revenue
- Sandwich and Smoothie are the next best performing items
- Payment methods contribute almost equally to total revenue
- In-store sales are higher than takeaway
- Monthly revenue trend remains relatively stable
